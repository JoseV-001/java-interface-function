# java-function-implementations
# Exemplos de Implementação de Function em Java

Este projeto contém diferentes formas de implementar a interface funcional `Function<T, R>`.

## Versões incluídas:

1. **Implementação via classe concreta**
2. **Method Reference com método estático**
3. **Method Reference com método não-estático**
4. **Expressão lambda declarada**
5. **Expressão lambda inline**
